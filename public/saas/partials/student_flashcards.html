Student Flashcards

The student flashcards can be created by uploading a student roster file. Please follow the guide line below to create and download the student flashcards.

Sample student roster Excel file: student_flashcards_example.xlsx

*Location name needs to be the same spelling as the location name of the SAAS system.

*The uploaded file needs to include the same fields in the same format as in the sample student roster Excel file.

*One file can be uploaded for each year.

*The old data will be overwritten after a new file is uploaded.
